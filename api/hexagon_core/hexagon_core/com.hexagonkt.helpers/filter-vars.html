filterVars

fun String.filterVars(parameters: Map<*, *>): String
Content copied to clipboard

Filter the target string substituting each key by its value. The keys format is: #{key} and all occurrences are replaced by the supplied value.

If a variable does not have a parameter, it is left as it is.

Return

The filtered text or the same string if no values are passed or found in the text.

Samples

import org.junit.jupiter.api.Test
fun main() { 
   //sampleStart 
   val template = "User {{user}}"
val parameters = mapOf<Any, Any>("user" to "John")

assert (template.filterVars(parameters) == "User John")
assert (template.filterVars() == template) 
   //sampleEnd
}
